ПЛАН ИСПРАВЛЕНИЯ: Раздел отзывов — пустой список (rvList)
==========================================================

ДАТА: 2026-04-19

СИМПТОМ:
  В разделе «Отзывы» (view-reviews) полностью пусто — не отображается
  ни индикатор загрузки, ни сообщение «Пока нет отзывов», ни ошибка.

КОРНЕВЫЕ ПРИЧИНЫ (3 штуки):

  1. ГЛАВНАЯ — отсутствие try/catch в initReviewsView():
     Функция rvRenderWriteSection() может выбрасывать исключение
     (например, при сложном innerHTML-шаблоне для авторизованных
     пользователей). Поскольку try/catch отсутствовал, исключение
     распространялось вверх, и rvLoadPage(1) НИКОГДА не вызывался.
     В результате rvList оставался с начальным контентом (пустым).

  2. ВИЗУАЛЬНАЯ — rvList стартовал пустым в HTML:
     <div id="rvList"></div> — до выполнения JS пользователь видит
     абсолютно ничего, даже если JS загружается с задержкой.

  3. КОНТРАСТНОСТЬ — цвет текста review-empty:
     color:#666 на фоне #0a0e14 (WCAG контраст ~2.5:1) — ниже
     нормы читаемости, особенно на мобильном экране.

ИСПРАВЛЕНИЯ В static/index.html:

  1. initReviewsView() — добавлен try/catch вокруг rvRenderWriteSection():
       try{ rvRenderWriteSection(); }catch(e){ console.warn(...); }
       rvLoadPage(1);  // вызывается ВСЕГДА, независимо от ошибок

  2. HTML rvList — добавлено начальное содержимое:
       <div id="rvList"><div class="review-empty">Загрузка отзывов...</div></div>

  3. CSS .review-empty — улучшена видимость:
       color: #666 → color: #999
       border: 1px dashed #161616 → 1px dashed #2a2a2a

  4. Текст ошибки в rvLoadPage — улучшена видимость:
       color:#444 → color:#777 для small-текста
